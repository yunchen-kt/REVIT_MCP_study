# 排煙窗檢討 Domain

## 概述

在 Revit 中自動檢討建築物排煙窗是否符合法規要求，涵蓋無開口樓層判定、無窗居室判定、排煙窗有效面積計算、改善建議、視覺化標註、Excel 報告匯出。

## 法規體系

排煙義務有兩套平行法規體系，必須同時滿足，取嚴格者：

| 體系 | 法源 | 觸發條件條文 | 設計規範條文 |
|------|------|-------------|-------------|
| **建築法** | 建築技術規則（設計施工編） | §100 | §101、§102 |
| **消防法** | 各類場所消防安全設備設置標準 | §28 | §188、§189 |

## 流程圖

```
輸入：樓層名稱、天花板高度來源
│
├── 入口 A（Step 1）：無開口樓層？
│   消防§4 + §28③
│   外牆有效開口 < 1/30 且 ≥ 1000m²
│
├── 入口 B（Step 3）：特定用途場所？
│   建技規§100① + 消防§28①
│   特定類別 ≥ 500m²
│
├── 入口 C（Step 5）：無窗居室？
│   建技規§1第35款第三目 + §100②
│   >50m² 且天花板下80cm通風面積 < 2%
│
│   任一入口觸發 ↓
│
├── Step 4：防煙區劃
│   每 500m² 以防煙壁區劃
│
├── Step 5：排煙窗計算
│   天花板下 80cm 內可開啟面積 ≥ 區劃面積 2%
│   水平距離 ≤ 30m
│
├── Step 6：改善建議
│   A. 固定窗改可開啟窗
│   B. 窗戶上移/加高
│   C. 新增窗戶
│   D. 改採機械排煙
│
└── Step 7：匯出 Excel 報告
```

## Step 1：無開口樓層判定

**工具**：`check_floor_effective_openings`

**法源**：消防設置標準 §4 + §28③

**有效開口條件**：
- 可內切直徑 ≥ 50cm 圓
- 下緣距地板 ≤ 120cm
- 面臨道路或 ≥ 1m 通路（加註人工確認）
- 無柵欄、玻璃厚 ≤ 6mm（從族群名稱推斷 + 加註）
- 十層以下另需至少 2 個大型開口（直徑 ≥ 1m 或 75cm × 120cm）

**判定**：有效開口面積 < 樓地板面積 1/30 → 無開口樓層

**後果**：≥ 1000m² 的無開口樓層須設排煙設備（§28③）

## Step 2+5 合併：排煙窗計算（含無窗居室判定）

**工具**：`check_smoke_exhaust_windows`

Step 2（無窗居室判定）與 Step 5（排煙窗計算）檢查的是同一件事：
天花板下 80cm 內可開啟通風面積是否 ≥ 地板面積 2%。

**法源**：
- 無窗居室：建技規 §1第35款第三目 → §100②
- 排煙設計：建技規 §101① + 消防 §188③⑦

**計算邏輯**：
1. 取得天花板高度（Room 參數 或 Ceiling 元素，由使用者選擇）
2. 定義有效帶：天花板高度 - 80cm ~ 天花板高度
3. 逐窗計算：
   - 窗頂若超過天花板 → 截斷至天花板
   - 帶內高度 = min(窗頂, 天花板) - max(窗台, 有效帶下緣)
   - 帶內面積 = 寬 × 帶內高度
   - 有效面積 = 帶內面積 × 開啟折減係數
4. 加總有效面積 ≥ 區劃面積 × 2% → 合規

**窗戶開啟方式判定表**（從族群名稱推斷）：

| 關鍵字 | 類型 | 折減係數 |
|--------|------|----------|
| casement, 平開, 側開, pivot, 樞軸, tilt, 內倒內開 | 全開型 | 1.0 |
| sliding, 橫拉, 推拉, hung, 上下拉 | 半開型 | 0.5 |
| awning, 上懸, 外推, hopper, 下懸 | 外推型 | 0.5 |
| louver, 百葉 | 百葉 | 0.5 |
| fixed, 固定, picture, 景觀 | 固定窗 | 0 |
| （其他） | 未知 | 0（加註需人工確認） |

**面積門檻**：> 50m² 的房間才需檢討（建技規 §1第35款第三目）

## Step 3：特定用途場所

不需新增工具，在 domain 邏輯層處理。

- 建技規 §100①：第69條第一、四類等，每層 > 500m²
- 消防 §28①：第12條第1款等場所，面積 ≥ 500m²

## Step 4：防煙區劃（初版簡化）

以「單一房間」作為最小排煙區劃單位。
- 房間面積 > 500m² → 標示警告「須以防煙壁分割」
- 防煙壁定義：自天花板下垂 50cm 以上不燃材料垂壁

## 視覺化

**上色**（自動）：
- 🟢 綠色：全開型窗（casement/pivot）— 有效
- 🟡 黃色：折減型窗（sliding/awning）— 有效但折減
- 🔴 紅色：固定窗 — 排煙無效
- ⚪ 灰色：未知 — 需人工確認

**檢討立面**（新工具）：
- `create_section_view`：建立面向外牆的剖面視圖
- `create_detail_lines`：畫天花板線、有效帶下緣線
- `create_filled_region`：有效帶範圍的半透明填充
- `create_text_note`：統計摘要文字
- `create_dimension`（既有）：窗戶寬高、帶內高度標註

## Step 8：§101 補充法規檢討

**法源**：建技規 §101（排煙設備之構造）

除排煙口面積（Step 5 已涵蓋）外，§101 尚有以下規定：

### 8-1：排風量規定（靜態提醒）

> 排風機應能隨排煙口之開啟而自動操作，其排風量**不得小於每分鐘一二○立方公尺**。

- **實作策略**：靜態提醒（不自動偵測）
- **原因**：排風機屬 MEP 設備，建築模型通常不建模；參數名稱各廠商不一致，強行偵測易誤判
- **報告呈現**：Excel Sheet 5 列為人工確認項目

### 8-2：中央管理室（半自動偵測）

> 建築物高度超過**三十公尺**或地下層面積超過**一千平方公尺**者，排煙設備之控制，應設置於中央管理室。

- **實作策略**：半自動偵測
- **偵測邏輯**：
  1. 計算建築物高度 = max(Level.Elevation) - min(Level.Elevation)
  2. 計算地下層總面積 = Σ(Elevation < 0 的樓層 Room 面積)
  3. 觸發條件：高度 > 30m **或** 地下面積 > 1000m²
  4. 搜尋 Room Name 含「中央管理室」/「防災中心」/「中控室」/「監控室」
  5. 判定：找到 → PASS（附房間資訊），沒找到 → WARNING
- **報告呈現**：Excel Sheet 5 含自動判定結果 + 房間資訊

## Excel 匯出

**工具**：`export_smoke_review_excel`

匯出 Excel (.xlsx)（使用 ClosedXML），包含五個工作表：
1. 樓層總覽（無開口樓層判定）
2. 房間排煙檢討明細
3. 窗戶明細
4. 改善建議
5. §101 補充法規檢討（排風量提醒 + 中央管理室偵測）

### Sheet 5：§101 補充檢討 欄位格式

**區塊一：建築物資訊**

| 欄位 | 說明 |
|------|------|
| 建築物高度 | max(Level) - min(Level)，單位 m，> 30m 顯示 ⚠ |
| 地下層總面積 | Σ(地下樓層 Room 面積)，> 1000m² 顯示 ⚠ |

**區塊二：地下層明細**（僅當有地下層時顯示）

| 欄位 | 說明 |
|------|------|
| 樓層 | 地下樓層名稱 |
| 標高(m) | Level Elevation（負值） |
| 面積(m²) | 該樓層 Room 面積加總 |
| 房間數 | 該樓層 Room 數量 |

**區塊三：檢討項目**

| 欄位 | 說明 |
|------|------|
| 檢討項目 | 排風量 / 中央管理室 |
| 法規依據 | §101 對應條文 |
| 觸發條件 | 該項目的觸發條件與計算結果 |
| 判定結果 | PASS（綠）/ WARNING（橘）/ MANUAL（藍）/ N/A（灰） |
| 說明 | 詳細判定說明或人工確認事項 |

**區塊四：中央管理室資訊**（僅當偵測到時顯示）

| 欄位 | 說明 |
|------|------|
| 房間名稱 | Room Name |
| 房間編號 | Room Number |
| 所在樓層 | Level Name |
| 面積 | Room Area (m²) |

**判定結果底色**：
- PASS → 淺綠 `#E2EFDA`
- WARNING → 淺橘 `#FFF3E0`
- MANUAL → 淺藍 `#E3F2FD`
- N/A → 淺灰 `#F2F2F2`
